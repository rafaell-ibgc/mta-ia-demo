# Demo — Atribuição Multi-Toque (MTA) com IA

Este projeto contém:
- src/mta.py: biblioteca com funções de simulação
- streamlit_app.py: app Streamlit
- data/mta_demo_dataset.xlsx: dataset fictício

## Como rodar localmente
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run streamlit_app.py
```
